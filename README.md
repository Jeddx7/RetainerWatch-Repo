# RetainerWatch (distribution)

This is a plugin **distribution** repo, not the source — it holds only the compiled plugin package (`latest.zip`) and the repo manifest Dalamud's plugin installer reads (`pluginmaster.json`).

## Installing

In Dalamud Settings → Experimental → Custom Plugin Repositories, add:

```
https://raw.githubusercontent.com/Jeddx7/RetainerWatch-Repo/main/pluginmaster.json
```

RetainerWatch will then appear in the plugin installer under that repository.
